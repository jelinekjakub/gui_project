
2. Navigate to the project directory:

    ```bash
    cd nsql_project
    ```

## Usage

1. Start Docker container:

    ```bash
    docker-compose build
    docker-compose up
    ```

2. Open your web browser and go to `http://localhost:5000` to access the application.
